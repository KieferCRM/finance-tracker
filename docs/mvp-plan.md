# MVP Plan

## Users
- Primary: tipped workers in restaurants and bars

## Core Screens
1. Auth
2. Dashboard (Tonight's Damage)
3. Add Shift Income
4. Add Expense
5. Monthly Report
6. Insights (Could've Saved)
7. Export CSV

## Metrics to Track
- total income (month)
- total expenses (month)
- savings amount
- savings rate
- category spend percentages

## Insights Rules (v1)
1. If discretionary spend > 25% of income -> flag high leak.
2. If savings entries == 0 in month -> suggest auto-save target.
3. If food+rides > 20% -> suggest per-shift cap.

## Revenue (v1)
- Free: current month + basic report
- Pro: unlimited history + advanced insights + export
